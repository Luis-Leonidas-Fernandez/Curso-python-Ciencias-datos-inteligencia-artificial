# 🐍 Curso de Python – Fundamentos

Bienvenido a este repositorio.

Aquí documentaré mi proceso de aprendizaje de **Python**, reuniendo ejercicios, prácticas y ejemplos desarrollados durante el curso.

## Objetivo

Construir una base sólida en los fundamentos del lenguaje Python mediante la resolución de ejercicios y la aplicación de buenas prácticas de programación.

## Contenido

A medida que avance el curso, este repositorio incluirá temas como:

* Variables y tipos de datos
* Operadores
* Entrada y salida de datos
* Estructuras condicionales
* Bucles
* Funciones
* Colecciones (listas, tuplas, conjuntos y diccionarios)
* Manejo de archivos
* Programación modular
* Ejercicios prácticos

## Propósito

Este repositorio funciona como un registro de aprendizaje y práctica continua, permitiendo seguir la evolución de los conocimientos adquiridos durante el curso.
